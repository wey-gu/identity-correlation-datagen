## Dataset Files

- `user.csv`: Name, Birth, Addr, Phone, Email
  - User: Name, Birth, Addr, Phone, Email
  - Phone: Number
  - Address: Address, GeoPosition
  - Email: email

- `loggin_record.csv`: IP, Device
  - IP: Addr, GeoPosition
  - Device: DeviceID
